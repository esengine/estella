## UIMask

将子实体裁剪到父级 `UIRect` 边界内。

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `enabled` | boolean | `true` | 是否启用裁剪 |
| `mode` | MaskMode | `Scissor` | `Scissor` (0)、`Stencil` (1) |

嵌套 `UIMask` 实体会裁剪到所有父级遮罩的交集区域。

<Aside type="caution">
  `UIMask` 只裁剪**子**实体——拥有 UIMask 的实体本身不会被裁剪。
</Aside>

## SafeArea

适配设备安全区域（刘海、圆角）。适用于移动端和微信小游戏。

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `applyTop` | boolean | `true` | 应用到顶部 |
| `applyBottom` | boolean | `true` | 应用到底部 |
| `applyLeft` | boolean | `true` | 应用到左侧 |
| `applyRight` | boolean | `true` | 应用到右侧 |
